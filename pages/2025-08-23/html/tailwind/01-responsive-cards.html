왜 Tailwind CSS인가?

  • ✓ Responsive Design: sm:, md:, lg: 접두사로 반응형 디자인을 쉽게 구현
  • ✓ Hover States: hover: 접두사로 인터랙티브한 효과를 인라인으로 추가
  • ✓ Flexibility: 유틸리티 클래스 조합으로 무한한 디자인 가능성
  • ✓ No Context Switching: HTML 파일에서 바로 스타일링 작업 가능
