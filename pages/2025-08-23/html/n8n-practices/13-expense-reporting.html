🔄 상세 자동화 프로세스

1

모바일로 영수증 스캔 (출장 중 실시간)

김대리님이 식사 후 카페에서 스마트폰으로 영수증 사진 촬영

시나리오: 부산 해운대의 한 식당에서 고객과 점심 식사 후, 김대리님이 회사 Slack 봇에 영수증 사진을 전송

📱 기존 방식
  • • 영수증을 지갑에 보관
  • • 출장 후 종이 영수증 정리
  • • 수기로 경비 신청서 작성
  • • 영수증 분실 위험
🤖 자동화 방식
  • • Slack 봇에 사진 전송 즉시
  • • OCR로 자동 데이터 추출
  • • GPS 위치 정보 자동 기록
  • • 클라우드에 안전 보관
2

AI 영수증 분석 및 데이터 추출 (30초 이내)

OCR과 AI가 영수증의 모든 정보를 자동으로 디지털화

시나리오: 업로드된 영수증 이미지를 Google Vision API가 분석하여 구조화된 데이터로 변환

🔍 수동 입력시
  • • 업체명 직접 타이핑
  • • 금액 계산기로 확인
  • • 날짜/시간 수기 입력
  • • 카테고리 수동 선택
  • • 평균 5분 소요
🧠 AI 자동 처리
  • • 업체명, 금액 자동 추출
  • • 날짜/시간 자동 인식
  • • 사업자번호 확인
  • • 카테고리 지능적 분류
  • • 30초 이내 완료
3

자동 정책 검증 및 승인 라우팅 (즉시)

회사 규정에 따른 자동 검증과 적절한 승인자에게 라우팅

시나리오: 15만원 점심 식사비가 회사 1인당 식비 한도(5만원)를 초과하여 팀장 승인이 필요한 상황

📋 기존 승인 과정
  • • 규정 확인을 위해 인사팀 문의
  • • 적절한 승인자 확인 필요
  • • 승인 요청서 별도 작성
  • • 이메일로 개별 전송
⚡ 자동 라우팅
  • • 규정 자동 대조 (고객 접대비로 분류)
  • • 김대리 → 팀장 → 부장 경로 설정
  • • Slack/이메일 동시 알림
  • • 모바일에서 원클릭 승인
4

승인 완료 후 회계 시스템 연동 (실시간)

최종 승인 즉시 ERP 시스템에 자동 전표 생성

시나리오: 부장님이 모바일에서 승인 버튼을 누르는 순간 모든 회계 처리가 자동 완료

📊 수동 회계 처리
  • • 회계팀이 승인된 경비 수집
  • • ERP에 수기 입력
  • • 계정과목 수동 분류
  • • 월말 일괄 처리로 지연
🔄 즉시 연동
  • • SAP ERP 자동 전표 생성
  • • 계정과목 자동 매핑
  • • 예산 잔액 실시간 차감
  • • 급여 시스템에 정산 데이터 전송
