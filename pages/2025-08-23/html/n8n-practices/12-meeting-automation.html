📋 실제 시나리오: 마케팅 팀 주간 회의

마케팅 팀 주간 성과 리뷰
📅 매주 월요일 오전 10시
👥 8명 참석
⏱️ 60분 예정
📍 A동 2층 회의실 + Zoom
진행 중

참석자: 마케팅 팀장 이수진, 디지털 마케터 김철수, 콘텐츠 기획자 박영희, 퍼포먼스 마케터 정민호, 브랜드 매니저 최유리, 디자이너 한지민, 데이터 애널리스트 오승환, 마케팅 인턴 신입이

주요 안건: 지난주 캠페인 성과 분석, 이번 주 마케팅 계획, 예산 검토, 크리에이티브 리뷰

🔄 자동화된 회의 라이프사이클

1
🎯 회의 계획 & 예약 (24시간 전)
  • 자동 일정 생성: 구글 캘린더에 반복 일정 설정
  • 참석자 초대: 팀 구성원 자동 초대 및 확인 요청
  • 회의실 예약: 사용 가능한 회의실 자동 검색 및 예약
  • Zoom 링크: 화상회의 링크 자동 생성 및 공유
2
📋 회의 준비 & 자료 수집 (2시간 전)
  • 안건 자동 생성: 이전 액션 아이템과 주간 데이터 기반
  • 자료 취합: GA4, 광고 계정에서 성과 데이터 자동 수집
  • 리마인더 발송: 참석자들에게 준비사항 알림
  • 사전 설문: 논의하고 싶은 주제 사전 수집
3
🎤 회의 진행 & 기록 (실시간)
  • 자동 녹음/녹화: Zoom 자동 녹화 및 클라우드 저장
  • 실시간 전사: AI 음성 인식으로 회의록 자동 생성
  • 참석 확인: 지각자 자동 알림 및 참석률 추적
  • 시간 관리: 안건별 소요시간 추적 및 알림
4
📝 회의록 정리 & 배포 (30분 후)
  • 회의록 요약: AI가 주요 내용과 결정사항 요약
  • 액션 아이템: 담당자별 할 일과 마감일 자동 추출
  • 자동 배포: 참석자와 관련 부서에 회의록 전송
  • 프로젝트 도구 연동: Jira, Notion에 태스크 자동 생성
5
📊 후속 조치 & 추적 (지속적)
  • 진행 상황 추적: 액션 아이템 완료 여부 자동 확인
  • 리마인더: 마감일 임박 시 담당자에게 알림
  • 성과 분석: 회의 효율성과 결과 지표 수집
  • 다음 회의 준비: 미완료 항목을 다음 안건에 자동 추가

🔍 상세 자동화 시나리오 분석

1

일요일 저녁 8시 - 주간 데이터 수집 시작

다음날 회의를 위한 자동 데이터 준비

상황: 시스템이 월요일 마케팅 회의를 24시간 앞두고 자동으로 준비 작업을 시작합니다.

🎯 트리거 조건
  • • 일요일 20:00 정시 실행
  • • 다음날 마케팅 회의 일정 확인
  • • 휴일 제외 로직 적용
⚡ 자동 실행 액션
  • • Google Analytics 주간 데이터 수집
  • • Facebook Ads, Google Ads 성과 추출
  • • 이메일 마케팅 지표 (Mailchimp) 수집
  • • CRM에서 리드 전환율 데이터 추출
  • • 자동 리포트 템플릿에 데이터 삽입
2

월요일 오전 8시 - 회의 준비 완료 알림

참석자들에게 회의 자료와 함께 리마인더 발송

상황: 회의 2시간 전, 모든 참석자에게 준비된 자료와 함께 상세한 안내 메시지를 전송합니다.

🎯 트리거 조건
  • • 회의 시작 2시간 전
  • • 참석 확인 미응답자 확인
  • • 회의실 예약 상태 점검
⚡ 자동 실행 액션
  • • 개인별 맞춤 리마인더 이메일
  • • 주간 성과 대시보드 PDF 첨부
  • • Slack에 회의 채널 생성
  • • Zoom 링크와 회의실 정보 공유
  • • 미응답자에게 개별 메시지
3

오전 10시 - 회의 시작과 실시간 관리

자동 녹화, 실시간 전사, 시간 관리

상황: 팀장 이수진님이 Zoom 회의를 시작하면서 모든 자동화 프로세스가 활성화됩니다.

🎯 트리거 조건
  • • Zoom 회의 시작 감지
  • • 참석자 3명 이상 입장
  • • 회의 호스트 권한 확인
⚡ 자동 실행 액션
  • • Zoom 클라우드 녹화 시작
  • • 실시간 음성-텍스트 변환
  • • 늦은 참석자 자동 알림
  • • 안건별 시간 추적 시작
  • • 화면 공유 내용 스크린샷 저장
4

오전 11시 - 회의 종료 후 즉시 처리

AI 기반 회의록 생성과 액션 아이템 추출

상황: 회의가 종료되자마자 AI가 녹음 파일을 분석하여 구조화된 회의록을 생성합니다.

🎯 트리거 조건
  • • Zoom 회의 종료 감지
  • • 녹화 파일 클라우드 업로드 완료
  • • 최소 30분 이상 진행된 회의
⚡ 자동 실행 액션
  • • GPT API로 회의록 요약 생성
  • • 액션 아이템과 담당자 자동 추출
  • • 다음 회의 일정 제안
  • • Notion/Jira에 태스크 생성
  • • 회의록을 참석자에게 자동 발송
